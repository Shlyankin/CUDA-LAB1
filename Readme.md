#Консольное приложение дл перемножени матриц на С++ с использованием GPU и CPU.
В программе присутствуют функции заполнения матриц, вывода в файл.
Проверка результата перемножения происходит в main функции.

######Перемножение матриц на CPU. 
Наиболее простейший алгоритм умножения матриц "в лоб".

######Перемножение Матриц на GPU. 
Использован алгоритм с использованием быстрого вида памяти shared memory для получения наиболее хорошего результата по времени.
Перемножение осуществляется блоками размера 16 на 16.

Вычисление производились на GTX 950 + intel core i5 2500
